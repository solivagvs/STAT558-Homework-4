  1. Lastly, we construct a function allowing a user to more easily query this API by topic, earliest_date, and API_key. For this problem, we will return a vector of the article titles.
news_search <- function(topic,
                        earliest_date = "2026-05-16",
                        API_key = "e7e98c8f31914c04a5d15115a45ea6e3"){
  
  format_query <- paste0("https://newsapi.org/v2/everything?q=",
                         topic,
                         "&from=",
                         earliest_date,
                         "&language=en&pageSize=50&apiKey=",
                         API_key)
  
  news_query <- GET(format_query)
  
  news_parsed <- fromJSON(rawToChar(news_query$content))
  
  news_parsed$articles$title
}
